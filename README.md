# Eventful Backend

Production-inspired backend for event management and ticketing.

## Stack

- Node.js
- TypeScript
- Express.js
- PostgreSQL
- Prisma
- JWT
- Redis
- BullMQ

## Features

- Authentication
- Event Management
- Ticket Booking
- QR Verification
- Analytics
- Reminders
- Paystack Payments

## Setup

```bash
npm install
```
